# Ricardo Fort API

[ricardofortapi](https://ricardofortapi.netlify.app/) es una API JSON gratuita para frases de Ricardo Fort seleccionados a mano.

## Uso

```shell
# Recupera una frase aleatoria del comandante en formato JSON.
GET https://ricardofort.herokuapp.com/

# Recupera todas las frases del comandante
GET https://ricardofort.herokuapp.com/all
```
Respuesta de ejemplo
```json
{
    "frase": "¡BASTA CHICOS!"
}
```
